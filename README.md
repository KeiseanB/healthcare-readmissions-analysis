# Florida Hospital Readmissions Analysis (FY2026)

## Overview
Analyzed CMS Hospital Readmissions Reduction Program (HRRP) data to identify 
Florida hospitals with excess Medicare penalties using Python and pandas.

## Key Findings
- 169 Florida hospitals analyzed
- 154 out of 169 (91%) are being penalized for excess readmissions
- Hip/Knee replacement procedures had the highest excess readmission ratios
- South Florida hospitals including Broward Health North and Delray Medical 
  Center among highest penalized

## Tools Used
- Python (pandas, matplotlib)
- CMS Public Dataset (data.cms.gov)
- Google Colab

## Data Source
FY2026 Hospital Readmissions Reduction Program — Centers for Medicare & 
Medicaid Services (CMS)
